## Installation

1. Clone the repository:
  `git clone https://github.com/vanbasx/wpdocker-classic`
2. Navigate to the work folder:
  `cd wpdocker-classic`
3. Build and Start Containers:
  `docker-compose up -d`
4. Open in browser:
   `http://localhost:8000`

To stop the running containers, use:
   `docker-compose down --volumes`
